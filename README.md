<h2>🛠️ Installation Steps:</h2>

<p>1. Install pnpm with your favorite package manager</p>

```
https://pnpm.io/installation
```

<p>2. Install dependencies</p>

```
pnpm i
```

<p>3. Run application on mobile</p>

```
pnpm android
```

<p>4. Run application on web</p>

```
pnpm web
```
